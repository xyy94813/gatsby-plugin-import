## Usage

### Install

Use npm: 
```
npm i gatsby-plugin-import -S
```

Or use yarn:
```
yarn add gatsby-plugin-import
```

### Config

in `gatsby-config.js`

```
module.exports = {
  // ...
  plugins: [
    // ...
    {
      resolve: 'gatsby-plugin-import',
      options: {
        libraryName: "antd",
        style: true,   // or 'css'
      }
    },
  ],
}
```

More usage see [there](https://github.com/ant-design/babel-plugin-import)
